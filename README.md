# api_text_matching

## Requirements
* python
* Node.js

## Setting Up
### Project
* clone project
> $ git clone https://github.com/thirdsos/api_text_matching.git

* use virtualenv or install packages globally
* pip install -r requirements.txt

### Build Database
> $ python TextCategory.py
> $ python unique_keyword.py
   
## Run Project
> $ python main.py
> $ cd web
> $ npm start
